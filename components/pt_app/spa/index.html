    Template & CSV data
    CSV data with field names in first row. Use variables… Use variables like {fieldname} in the label text to substitute.

    Optional $copies column repeats a row N times in batch print. A signed integer in a second |-segment shifts any numeric value (or seconds, for date helpers).

    {fieldname}
    CSV column lookup
    {$dt|YYYY-MM-DD}
    current date
    {$dt|YYYY-MM-DD|+86400}
    tomorrow (offset in seconds)
    {$time|HH:mm}
    current time
    {$now}
    ISO date+time
    {$row}
    0-based data-row index, header excluded
    {$row|+1}
    same +1, i.e. 1-based row number
    {$copy}
    0-based copy index, varies only when $copies > 1
    {$seq}
    1-based label number in batch
    {$total}
    total labels in batch

    Wi-Fi

    Pick a network and enter the password. Saving reboots the device into the new network. If it can't connect, the next boot brings up the pt700-setup AP for re-onboarding.

    API host

    When the SPA is loaded outside the device (localhost dev server), set this to the device URL and the page will hit it cross-origin. The embedded build defaults to same-origin and ignores this.

    Language

    Additional locales load on demand from i18n/{lang}.json. New translations get added to the dropdown by extending the STRINGS table or shipping a JSON file.
