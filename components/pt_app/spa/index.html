Settings

When the SPA is loaded outside the device (localhost dev server), set API host to the device URL and the page will hit it cross-origin. The embedded build defaults to same-origin and ignores this. Some browsers block file:// → http:// fetch — serve the spa/ directory with python3 -m http.server if so.
